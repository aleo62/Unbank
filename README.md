# Unbank

Unbank é um projeto educacional que simula o funcionamento básico de um banco digital de forma satírica.  
O nome utiliza o prefixo “un”, do inglês, para deixar claro que este **não é um banco real**.

Este projeto foi criado exclusivamente para fins de estudo e portfólio.

---

## Objetivo do Projeto

O objetivo do Unbank é:
- Praticar desenvolvimento backend com Java e Spring Boot
- Construir uma API REST seguindo boas práticas
- Integrar um frontend em React utilizando Vite
- Simular regras comuns de sistemas bancários
- Servir como projeto de portfólio

---

## Arquitetura

O projeto é dividido em duas aplicações independentes:

### Backend
- Desenvolvido em Java com Spring Boot
- Responsável pelas regras de negócio
- Exposição de uma API REST
- Camadas bem definidas (controller, service, repository)

### Frontend
- Desenvolvido em React com Vite
- Aplicação SPA
- Consumo da API REST via HTTP
- Interface simples e focada em demonstração

---

## Funcionalidades

- Cadastro de usuários
- Criação de contas
- Consulta de saldo
- Simulação de transferências
- Histórico de operações

As funcionalidades podem evoluir conforme o projeto avança.
